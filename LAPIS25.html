Hands-on activities - Day 1

*This material was adapted from the S-PLUS workshop held in SAB 2024, and presented in XI LAPIS 2025. Prepared in collaboration with Eduardo Lacerda (IAG/USP), Gustavo Schwarz (Poli/USP) and Lilianne Nakazono (IF/USP)
The proposed exercise here will guide you through two simple examples using the S-PLUS DR4. For the execution of this hands-on, you will follow some of the documentation and code examples given during the previous lectures. In particular, we will use LSDB, splusdata and some common Python libraries like NumPy (for calculations) and Matplotlib (for plots). In addition to the mentioned technical tools, the exercise require data from the the S-PLUS DR4, associated VACs and the GAIA DR3 catalog.

Proposed science cases:

  1. Create a HR Diagram for a globular cluster identified in S-PLUS DR4/PSF catalog, crossmatching with GAIA DR3

  2. Create an observational CMD (e.g., g-r vs r) diagram for galaxies, and choose one to create an IFS-like "SCUBE"

  3. Create an observational CMD (e.g., g-r vs r) diagram for galaxies, and choose one to create an IFS-like "SCUBE"
